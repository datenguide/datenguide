# Verwendung mit Javascript

Im Folgenden findest du Beispiel-Code, um die REST-Schnittstelle mit JavaScript abzufragen. Eine Schritt-für-Schritt-Anleitung wie die URL für die Abfrage der Schnittstelle aufgebaut sein muss, findest du unter [Aufbau einer REST-Abfrage](abfragen).

Um die Daten zu abzufragen, benutzen wir die [Fetch API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API/Using_Fetch), die in modernen Browsern zur Verfügung steht.

```js
// Basis-URL der Datenguide API:
const BASE_URL = 'https://tabular.genesapi.org'

// Einfache Funktion zur Herunterladen und Anzeigen
// eines Datensatzes im JSON-Format:
function getData(query) {
  return fetch(BASE_URL + '?format=json&' + query)
    .then((response) => response.json())
    .then((result) => {
      console.log('Data:', result.data)
    })
    .catch((error) => {
      console.error('Error:', error)
    })
}

// Beispielhafter Funktionsaufruf für:
// - eine Statistik ('11111' / Gebietsstand)
// - ein Wertmerkmal ('FLC006' / Fläche)
// - eine Region ('11' / Berlin)
getData('data=11111:FLC006&region=11')
```
