# Dauerhafte Links

Wir gleichen unsere Daten regelmäßig mit denen der statistischen Landesämter ab und halten damit unsere 
Datenbasis auf dem neuesten Stand. Das heißt aber auch, dass die gleichen Abfrage-Links für die 
REST-Schnittstelle unterschiedliche Daten liefern können.

Wir bieten dir mit https://static.tabular.genesapi.org ein Tool an, das genauso funktioniert wie 
https://tabular.genesapi.org – nur, dass die hier generierten Tabellen nicht verändert werden und so die 
Links dauerhaft erreichbar bleiben.

####### Geht es um die Dauerhafte Erreichbarkeit der Links oder die Unveränderlichkeit der Ergebnisse?

## Abfragen über die "normale" REST-Schnittstelle

Alle Einbürgerungen von Ausländern mit Aufenthaltsdauer von unter 8 Jahren in allen brandenburgischen Landkreisen ausgeben, im Zeitraum seit 2000:

####### LINKS EINFÄRBEN? ##### 

```js
https://tabular.genesapi.org/?region=all&level=3&parent=12?time=2000:?data=12511:BEV008(AUFDA1:AUFDA00B08)
```

## Dauerhaft erreichbare Abfrage-Links erstellen

Damit diese Abfrage auch in zwei Jahren noch erreichbar ist, kann sie über die Subdomain _static_ gestellt 
werden:

```js
https://static.genesapi.org/?region=all&level=3&parent=12?time=2000:?data=12511:BEV008(AUFDA1:AUFDA00B08)
```
