## Über Datenguide

Die **Statistischen Ämter** sammeln kontinuierlich Daten und erstellen Statistiken zu allen Bereichen unserer Gesellschaft. Das Ziel von Datenguide war es, diese **amtliche Statistik** zugänglicher zu machen.

Von **2018 bis 2024** haben wir bei Datenguide an Open&#8209Source&#8209;Tools gearbeitet, die halfen, die Daten der amtlichen Statistik zu verstehen und sinnvoll einzu&shy;setzen. Wir betrieben unter anderem eine **[API](/docs)** und ein experimentelles **[Datenportal](/statistiken)** für regionale Statistiken. Diese Angebote sind inzwischen eingestellt.

</Intro>

## Open Source (archiviert)

Datenguide bot über eine Datenschnittstelle (API) direkten Zugriff auf Daten der deutschen Regionalstatistik. **Diese API ist seit Juni 2024 abgeschaltet**. Die Daten können über die [Regionaldatenbank Deutschland](https://www.regionalstatistik.de/genesis/online/) abgerufen werden, die inzwischen eine offizielle API anbietet.

Auf die API von Datenguide konntet ihr über [REST](https://github.com/datenguide/genesapi-tabular) und [GraphQL](https://github.com/datenguide/datenguide-api) zugreifen, sowie über Bibliotheken für Python und R, die von der [CorrelAid-Community](https://correlaid.org/) entwickelt wurden. **Diese Tools werden nicht mehr weiterentwickelt.** Der Quellcode ist auf GitHub archiviert.

</Tools>
